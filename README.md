## Technologies Used

- **Language:** Python 3.x
- **NLP & ML:** Scikit-learn, XGBoost, Transformers (HuggingFace), SpaCy, NLTK
- **Big Data:** PySpark
- **Visualization:** Matplotlib, Seaborn
- **Notebook:** Jupyter
- **Data:** Reddit, Twitter, and GoEmotions datasets
